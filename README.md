# 💧 Water Reminder 

This is a simple Python project that reminds users to drink water at regular intervals using desktop notifications.

## 🚀 Features
- Sends notification every hour
- Helps maintain hydration
- Lightweight and easy to use
- Runs continuously in the background

## 🛠️ Technologies Used
- Python
- time module
- plyer (for notifications)

## 📦 Installation

1. Clone the repository:
   git clone https://github.com/your-username/water-reminder.git

2. Install required library:
   pip install plyer

3. Run the program:
   python water_reminder.py

## ⏰ How it works
The program runs an infinite loop and sends a notification every 60 minutes reminding the user to drink water.

## 💡 Use Case
This project is useful for students, developers, and office workers who often forget to stay hydrated while working.
